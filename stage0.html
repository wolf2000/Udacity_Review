The basics of the Web

HTML
HyperText Markup Language - Main document type on the Web

URLs
Uniform Resource Locator - reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it

HTTP
the main protocol of the web

web applications
software application which the client runs in a web browser

Servers
computers that host the files that make up the web

Browser
A program that runs on your computer to display files found on the web

HTML Tags

Bold Tag
<b>contents</b>

Italics
em- emphasis - italics
<em>contents</em>

Making Links
<a href="url">contents</a>

Adding images
<img src="url" alt="text">

whitespace
<br> break (inline elements)

paragraph
<p>content</p> (block elements)

Span and Div
Span - inline Container elements <span class="foo">text</span>
Div - block Container elements <div class="bar">text</div>

HTML Documents structure

<!DOCTYPE HTML> - document type is HTML
<html> </html> - html tag surrounds the entire rest of document.
<head> </head> - Metadata, Javascript , css included in the head section.
<title>Title!</title> - title of the page
<body> </body> - body tag is actual contents of the document.
